# Medical Website
website wherein the users can get the latest medical news and, and book an appointment for a doctor working there.

### Technologies
#### Front-End
* HTML / CSS 
* jQuery
* JavaScript
* AOS (Animate on scroll library)
 #### Back-End
 * AJAX
 * Public API 


## API
 API i have used is : public api (Healthcare.gov)
